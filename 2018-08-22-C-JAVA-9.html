1.背景介绍

什么是spring boot

随着Spring的功能越来越强，Spring本身变得越来越臃肿。即使只是完成一个发送邮件的小项目也需要配置一堆的配置文件。 SpringBoot简化了Spring应用开发，

2.知识剖析

springboot的优点

1.实现约定大于配置，是一个低配置的应用系统框架。不像Spring那样“地狱般的配置体验”，SpringBoot不需要或极少配置，就能使用大量的功能;

2.提供了内置的Tomcat容器;

3.通过依赖的jar包管理、自动装配技术，容易支持与其他技术体系、工具集成；

4.支持热加载，开发体验好。
